FinEase - Personal Finance Management App

Live Site URL: https://finease-personal-finance.netlify.app/  

Features:
1. Users can add, update, view, and delete transactions easily.
2. Separate pages for Transaction Update and Transaction Details.
3. Each user sees only their own transactions using email-based filtering.
4. Smooth UI with instant delete update (UI refresh without reload).
5. Transaction details page shows total amount of the same category.
6. Secure routes protected using PrivateRoute authentication.
7. Clean and responsive UI using React + Tailwind CSS.

Technologies Used:

1. React.js  
2. React Router  
3. Firebase Authentication  
4. Tailwind CSS  
5. MongoDB + Express.js Backend  
6. SweetAlert2  
7. Context API
